## Basic Overview
Chefs Journal is a web app, inspired by my pandemic hobby, where I created recipes and shared them on instagram. 
This is a full stack project presented as my capstone project at Brainstation. The idea was to challenge myself and create my own database and API in addition to the front end work.
View the deployed version here: <a href="https://chefsjournal.ca">chefsjournal.ca</a>
